Dúvidas frequentes

FAQ

Funciona para vendedores com CPF?

Sim. A calculadora tem opções específicas para CPF com menos de 450 pedidos e CPF com mais de 450 pedidos em 90 dias, aplicando as taxas corretas para cada perfil.

Funciona para vendedores CNPJ?

Sim. Basta selecionar "CNPJ" no tipo de vendedor. As faixas de comissão e taxas fixas são aplicadas automaticamente conforme a tabela oficial da plataforma.

Preciso instalar algum aplicativo?

Não. A calculadora roda direto no navegador — basta abrir o link. No celular, você pode adicionar à tela inicial como um app (PWA) com um toque, sem precisar da loja de aplicativos.

Funciona no celular?

Sim, 100%. O layout é totalmente otimizado para celular. Você pode instalar na tela inicial do iPhone ou Android para abrir como um aplicativo nativo, sem internet para acessar.

Como recebo o acesso PRO após a compra?

Após o pagamento, você recebe um e-mail com uma senha de ativação. Basta digitar a senha dentro da própria calculadora para liberar os cálculos ilimitados imediatamente — sem esperar.

Pronto para calcular sem limite?

Uma compra única. Sem mensalidade. Cálculos ilimitados para sempre.
